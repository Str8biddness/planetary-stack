⚙️ Settings
Result verification
Require verified evidence
Refuse a result unless the machine that produced it signed a valid execution record. Turn this off and results are still checked and still labelled — you just are not blocked.
What a verified result means. The machine that ran the work signed a record naming the image, entrypoint, inputs and lease it used, and this desktop checked that signature against the key it recorded when the device was enrolled. It proves which machine produced the result and that the approved image and inputs were named — it does not prove that machine is honest. A compromised device holds its own key. Hardware attestation is not available yet.
